# Technical Proposals, RFCs, and Design Sessions

Supports drafting technical documentation, system design, architecture, and RFCs with structured thinking.

## Process

0. Clarify objective and scope
1. Map stakeholders and constraints
2. Generate multiple approaches
3. Compare via decision matrix
4. Write proposal with: context, problem, options, recommendation
5. Validate E2E and verify
